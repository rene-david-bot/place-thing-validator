What this PoC validates

Phase 1, place memory

  • Map workshop, cellar, shelves, drawers, cabinets, benches
  • Capture multiple views with deliberate variance
  • Create a reusable place memory index

Phase 2, object in context

  • Capture an object where it lives
  • Use object + background to infer likely place
  • Test retrieval, confidence, and trust

Capture checklist

Place capture guidance

  • Take one wide overview first.
  • Move slightly left or right for the second angle.
  • Capture the surrounding context, not just a neat close-up.
  • Add one realistic messy shot if this place is often cluttered.
  • Avoid nearly identical photos.

Stage 2 guidance

  • Keep the object clearly visible.
  • Do not crop out the surrounding location.
  • If two objects are similar, include a second angle.
  • Trust drops fast when the model sounds too certain, so log ambiguity.
